# Automated API Testing & Security Verification

This project focuses on automated validation of REST API endpoints, response data integrity, and fundamental security header checks using Python.

## Scope of Testing
- **Functional Testing:** Verifying status codes (200 OK, 201 Created).
- **Data Integrity:** Ensuring transmitted payloads match server responses.
- **Security Check:** Verifying response headers and payload format.

## Execution
1. Install requirements: `pip install requests`
2. Run tests: `python test_api.py`
